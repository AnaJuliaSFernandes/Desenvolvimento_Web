O que é Bootstrap

O Bootstrap é um fremework de código aberto e gratuito, utilizado para desenvolvimento front-end fornecendo componentes pré-construidos e um sistema de grid responsivo que permite o desenvolvimento de aplicação html e sites com mais facilidade
